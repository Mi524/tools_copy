一、 
 1-整体部分：涉及输入的BI售后数据+意见反馈+呼叫中心数据，脚本会从输入的BI售后数据中截取日期的最后一个月生成“提取日期.txt”文档，此文档是默认的 数据提取日期（即如果单纯提取之前已经计算过的数据，可以直接在提取日期修改想要的日期)

二、
 2-写入故障现象表：涉及输入的“故障现象数量"文件夹，里面放的所有文档都会被读取写入 
 （如果不小心写入了重复的历史数据，需要在库里删掉重新写，谨慎不要放入历史文档）

三、
 3-运行计算程序：对写入库里的数据计算，全程大概需要10分钟（曾经算过的数据不需要再重新运行该步骤脚本）

四、
 4-提取计算结果和分析结果（这个月已经计算过的数据可以直接运行提取）

五、
 5-调用VBA画图，最好关闭EXCEL再运行脚本，否则画图期间EXCEL界面会闪来闪去

requirements

pip install -i https://pypi.doubanio.com/simple/ pandas==1.0.1 xlsxwriter==1.2.9 sqlalchemy==1.3.17 pypiwin32 mysqlclient==1.4.6


检查电脑是否装有macrosoft visual C++ 2015 Redistributable 以上版本(64位操作系统X64,32位操作系统X86)
检查电脑对于EXCEL的注册表指向的是WINDOWS EXCEL,而不是WPS


1.cmd regedt32
2.在注册表中查看[HKEY_CLASSES_ROOT\Excel.Application]项
3.在注册表中搜索 00024500-0000-0000-C000-000000000046 文件夹

主要看 LocalServer, LocalServer32是否被WPS修改成了WPS的启动项位置，修改成正确的默认Macrosoft Excel位置
格式 C:\PROGRA~1\MICROS~1\Office15\EXCEL.EXE /automation

画图如果提示 不信任的宏 在EXCEL选项，宏-->信任任何来源的代码调用宏 打钩

修改注册表和修改EXCEL设置之后可能都需要重启电脑（最好是关机，再重新开机）
